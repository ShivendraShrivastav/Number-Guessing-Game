# Number-Guessing-Game
Guess The Random Number Between 1 To 100 Generated By The Computer

A simple command-line Number Guessing Game written in C. The program generates a random number between 1 and 100, and the player has to guess it with the help of hints.

📌 Features
Generates a random number between 1 and 100
Provides hints:
Go Higher if the guess is too low
Go Lower if the guess is too high
Counts the number of attempts taken to guess the correct number
Uses the C Standard Library's random number generator
🛠️ Technologies Used
C Programming Language
Standard Libraries:
stdio.h
stdlib.h
time.h

▶️ How to Compile and Run
Using GCC

Compile the program:
gcc main.c -o guess

Run the executable:
./guess

On Windows (MinGW)
gcc main.c -o guess.exe
guess.exe

🎮 How to Play
Run the program.
Enter a number between 1 and 100.
The program will respond with:
Go Higher if your guess is too low.
Go Lower if your guess is too high.
Keep guessing until you find the correct number.
The program will display how many attempts you took.

📖 How It Works
Seeds the random number generator using the current system time.
Generates a random number from 1 to 100.
Repeatedly asks the user for a guess.
Compares the guess with the generated number.
Displays hints until the correct number is guessed.
Counts and displays the total number of attempts.
